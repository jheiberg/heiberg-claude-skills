# Konsole Shortcuts

Config: `~/.config/konsolerc`  
No custom keybindings configured — all shortcuts are Konsole defaults.

## Tabs

| Key | Action |
|-----|--------|
| `Ctrl+Shift+T` | New tab |
| `Ctrl+Shift+W` | Close tab |
| `Shift+Left` / `Shift+Right` | Switch to previous / next tab |
| `Ctrl+Alt+1–9` | Switch directly to tab N |
| `Ctrl+Shift+Left` / `Ctrl+Shift+Right` | Move tab left / right |
| `Ctrl+Alt+S` | Rename current tab |

## Split view (multiple terminals in one window)

| Key | Action |
|-----|--------|
| `Ctrl+Shift+(` | Split view left/right (side by side) |
| `Ctrl+Shift+)` | Split view top/bottom (stacked) |
| `Ctrl+Shift+Z` | **Maximize / restore focused pane** (toggle) |
| `Ctrl+Shift+[` | Close active terminal pane |
| `Ctrl+Tab` | Move focus to next terminal pane |
| `Ctrl+Shift+Tab` | Move focus to previous terminal pane |

## Copy & paste

| Key | Action |
|-----|--------|
| `Ctrl+Shift+C` | Copy selection |
| `Ctrl+Shift+V` | Paste |
| Middle-click | Paste X clipboard (selected text) |

## Scrollback & search

| Key | Action |
|-----|--------|
| `Ctrl+Shift+F` | Find in scrollback |
| `Shift+Up` / `Shift+Down` | Scroll up / down one line |
| `Shift+PgUp` / `Shift+PgDn` | Scroll up / down one page |
| `Ctrl+Shift+Home` | Scroll to top of scrollback |
| `Ctrl+Shift+End` | Scroll to bottom |

## Window & zoom

| Key | Action |
|-----|--------|
| `Ctrl+Shift+N` | New Konsole window |
| `Ctrl++` | Increase font size |
| `Ctrl+-` | Decrease font size |
| `Ctrl+0` | Reset font size |
| `Ctrl+Shift+F11` | Toggle fullscreen |
| `Ctrl+Shift+M` | Toggle menu bar |

## Profile & settings

| Key | Action |
|-----|--------|
| `Ctrl+Shift+,` | Edit current profile |
